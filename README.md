# Godot Mobile Tap Game

간단한 모바일 탭 게임입니다. 

## 게임 방법
- 화면을 터치하여 플레이어(초록색 사각형)를 좌우로 이동
- 위에서 떨어지는 적(빨간색 사각형)을 피하세요
- 적을 피할 때마다 점수가 올라갑니다
- 시간이 지날수록 게임 속도가 빨라집니다

## 실행 방법

1. Godot 4.3 이상 설치
2. Godot에서 프로젝트 열기
3. Main.tscn 씬 실행

## 모바일 빌드

### Android
- Android Export Templates 설치 필요
- Project > Export > Android 선택 후 APK 빌드

### iOS  
- Xcode 필요
- Project > Export > iOS 선택 후 빌드
